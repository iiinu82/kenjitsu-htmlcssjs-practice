# 杉山株式会社 - 堅実的な企業サイト

**信頼感と落ち着きのあるミニマル企業コーポレートサイト**（練習サンプル）

書籍『動くWebデザインアイディア帳』を参考に作成した、人材派遣会社のコーポレートサイトです。  
ローディングアニメーション、スクロールでヘッダー出現/消失、ゆっくりズームアウトのメインビジュアル、ランダム登場ボタン、タイムラインの線アニメーションなど

## デモ（ライブプレビュー）
https://kenjitsu-htmlcssjs-practice.vercel.app

## 主な実装機能
- **ローディングアニメーション** - ロゴ浮き上がり → 背景が上下に別れて消える
- **メインビジュアル**（Vegas.js） - ゆっくりズームアウト + スライド切り替え
- **スクロールでヘッダー出現/消失** - 下スクロール中に一定箇所で上に隠れる
- **右から/左からのスライド登場**（fadeRightTrigger / fadeLeftTrigger）
- **ランダム順でボタン登場**（randomScroll） - 事業紹介エリア
- **タイムライン線アニメーション** - Historyエリアでスクロールに合わせて伸びる
- **スマホ用ハンバーガーメニュー** - 右上ボタンで円形背景拡大 + メニュー出現
- **ページトップボタン** - フッターに付属。クリックでpage-topに移動

## 使用技術・ライブラリ
- HTML5 / CSS3
- jQuery 3.4.1
- [Vegas.js](https://vegas.jaysalvat.com/) - メインビジュアルスライダー + ズームアニメ
- [jQuery Easing](https://jqueryui.com/easing/) - スムーススクロール速度変化
- Google Fonts（Alex Brush + Noto Serif JP）- (タイトル筆記体Alex Brush)
